AI Travel Buddy — Mobile Web App MVP

A sleek, mobile-first AI-powered travel companion that helps users discover trips, generate itineraries, and receive live, context-aware travel guidance. Designed to feel like a real mobile app with a modern neon-glow aesthetic and smooth navigation.

🚀 Overview

AI Travel Buddy is a progressive, mobile-first travel planner that blends beautiful UI with practical features.
Users can browse curated trip ideas, generate personalized itineraries with natural language, and even view a live “travel buddy” screen with distance/time indicators and guidance.

This MVP is built as a single-file HTML application, ideal for demos, validation, portfolio use, or as a base for further development.
